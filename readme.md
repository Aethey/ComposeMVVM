# Jetpack Compose Simple GitHub App

## Description

A simple GitHub client built using Jetpack Compose and follows the MVVM

- add api_key in local.properties then build

## Architecture

- MVVM

## Feature

- list user.
- search user.
- list user repositories without fork.
- save search history.

## Incomplete

- unit test.
